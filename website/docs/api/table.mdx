## table

```ts
table(headers: string[], rows: string[][], options?: TableOptions): string
```

<Field name="headers" type="string[]">Column header labels. Rendered in bold.</Field>
<Field name="rows" type="string[][]">Data rows as arrays of cell values.</Field>

Renders a table using box-drawing characters.

```ts
const result = table([
  ['Name', 'Version', 'Status'],
  ['boltdocs', '2.8.0', '✅'],
  ['dui', '0.1.2', '✅'],
])
console.log(result)
```

### Options

<Field name="style" type="'single' | 'double' | 'round' | 'none'" default="'single'">Border style. `'none'` uses 2-space gaps.</Field>
<Field name="width" type="number" default="auto">Table width. Defaults to terminal width.</Field>
<Field name="columns" type="TableColumnOptions[]">Per-column alignment config.</Field>
<Field name="headerSeparator" type="boolean" default="true">Draw a separator line after the header row.</Field>
<Field name="padding" type="number" default="1">Inner cell padding.</Field>
<Field name="color" type="ColorInput">Border color for the table outline and header separator. Accepts hex, rgb, rgba, oklch, or named color.</Field>

### TableColumnOptions

<Field name="align" type="'left' | 'center' | 'right'" default="'left'">Text alignment for this column.</Field>

<Callout variant="info">
  Column widths auto-compute from content, then shrink proportionally if total exceeds available width.
</Callout>
