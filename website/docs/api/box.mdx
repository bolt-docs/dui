## box

```ts
box(lines: string[], opts?: BoxOptions): string
```

Generic box builder. Width adapts to terminal size by default.

```ts
box(['Line 1', 'Line 2'], {
  title: 'Status',
  style: 'double',
})
```

### shorthands

```ts
double(['Content'], { title: 'Title' })
single(['Content'], { title: 'Title' })
round(['Content'], { title: 'Title' })
```

### Options

<Field name="title" type="string">Bold title centered in the top border.</Field>
<Field name="width" type="number" default="auto">Box width. Defaults to terminal width (capped at 80).</Field>
<Field name="style" type="'single' | 'double' | 'round'" default="'single'">Border style.</Field>
<Field name="padding" type="number" default="1">Inner horizontal padding.</Field>
<Field name="color" type="ColorInput">Border color. Accepts hex, rgb, rgba, oklch, or named color.</Field>

All box shorthands also accept a `color` option to tint the border lines:

```ts
round('Notice', ['Server starting...'], { color: '#ff8800' })
```
