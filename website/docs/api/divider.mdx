## divider

```ts
divider(char?: string, len?: number): string
```

<Field name="char" type="string" default="'в”Ђ'">Repeat character for the line.</Field>
<Field name="len" type="number" default="terminal width">Line length in columns.</Field>

Returns a horizontal line that fits the terminal width.

### Default

```ts
divider('в”Ђ')        // gray solid line
```

## dividerLog

```ts
dividerLog(char?: string, len?: number, opts?: { color?: ColorInput }): void
```

Prints the divider directly to stdout.

```ts
dividerLog()
dividerLog('В·', 16)
dividerLog('в•ђ', 40, { color: '#00cc88' })
```
