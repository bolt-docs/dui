## createSpinner

```ts
createSpinner(message: string, opts?: SpinnerOptions): Spinner
```

<Field name="message" type="string">Initial status text.</Field>
<Field name="opts.prefix" type="string" default="configured prefix">Custom prefix for this spinner.</Field>
<Field name="opts.color" type="ColorStyle">Foreground and/or background color for the spinner frames and status icon. Supports hex, rgb, rgba, oklch, or named colors.</Field>

Creates an animated spinner for long-running operations.

### usage

```ts
const spinner = createSpinner('Installing dependencies...')
spinner.start()

// ... do async work ...

spinner.stop('success', 'Installed!')
spinner.stop('fail', 'Build failed!')
```

### api

<Field name="start" type="() => void">Start the spinner animation.</Field>
<Field name="update" type="(msg: string) => void">Update the displayed message.</Field>
<Field name="stop" type="(status?, msg?) => void">Stop with a status icon (`'success'`, `'fail'`, `'warn'`, or `'info'`).</Field>

| Status | Icon |
|--------|------|
| `'success'` | green ✔ |
| `'fail'` | red ✖ |
| `'warn'` | yellow ⚠ |
| `'info'` | blue ℹ |

Override the spinner color by passing `color` in the options:

```ts
const spinner = createSpinner('Loading...', { color: { fg: '#00aaff' } })
spinner.start()
```
