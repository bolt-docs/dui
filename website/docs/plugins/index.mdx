**DUI** has a lightweight plugin system that lets you extend the core with custom
functionality. These are the official plugins maintained by the DUI team.

---

## Official plugins

<Cards cols={2}>
  <Card title="@dui-toolkit/plugin-markdown" icon={<File />} href="/docs/plugins/markdown">
    Render markdown with colors, tables, and syntax-highlighted code blocks.
  </Card>
  <Card title="@dui-toolkit/plugin-chart" icon={<BarChart3 />} href="/docs/plugins/chart">
    Bar, column, line, pie, and sparkline charts for your terminal.
  </Card>
  <Card title="@dui-toolkit/plugin-image" icon={<Image />} href="/docs/plugins/image">
    Render images — PNG, JPG, and GIF — in the terminal with ANSI half-block art and Kitty protocol.
  </Card>
  <Card title="@dui-toolkit/plugin-diff" icon={<GitCompare />} href="/docs/plugins/diff">
    Unified, side-by-side, and word-level diffs themed for the terminal.
  </Card>
  <Card title="@dui-toolkit/plugin-qrcode" icon={<QrCode />} href="/docs/plugins/qrcode">
    Scannable QR codes in the terminal with custom colors, width cap, and labels.
  </Card>
</Cards>
