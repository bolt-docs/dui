**DUI** tiene un sistema de plugins ligero que permite extender el núcleo con
funcionalidad personalizada. Estos son los plugins oficiales mantenidos por el
equipo de DUI.

---

## Plugins oficiales

<Cards cols={2}>
  <Card title="@dui-toolkit/plugin-markdown" icon={<File />} href="/docs/es/plugins/markdown">
    Renderiza markdown con colores, tablas y código resaltado.
  </Card>
  <Card title="@dui-toolkit/plugin-chart" icon={<BarChart3 />} href="/docs/es/plugins/chart">
    Gráficos de barras, columnas, líneas, pie y sparkline para tu terminal.
  </Card>
  <Card title="@dui-toolkit/plugin-image" icon={<Image />} href="/docs/es/plugins/image">
    Renderiza imágenes — PNG, JPG y GIF — en la terminal con ANSI half-block art y protocolo Kitty.
  </Card>
  <Card title="@dui-toolkit/plugin-diff" icon={<GitCompare />} href="/docs/es/plugins/diff">
    Diffs unificados, lado a lado y a nivel de palabra, con tema de terminal.
  </Card>
  <Card title="@dui-toolkit/plugin-qrcode" icon={<QrCode />} href="/docs/es/plugins/qrcode">
    Códigos QR escaneables en la terminal con colores, tope de ancho y etiquetas.
  </Card>
</Cards>
