## divider

```ts
divider(char?: string, len?: number): string
```

<Field name="char" type="string" default="'в”Җ'">CarГЎcter repetido para la lГӯnea.</Field>
<Field name="len" type="number" default="ancho terminal">Longitud de la lГӯnea en columnas.</Field>

Devuelve una lГӯnea horizontal que se ajusta al ancho de la terminal.

### Por defecto

```ts
divider('в”Җ')        // lГӯnea sГіlida gris
```

## dividerLog

```ts
dividerLog(char?: string, len?: number, opts?: { color?: ColorInput }): void
```

Imprime el divisor directamente en stdout.

```ts
dividerLog()
dividerLog('В·', 16)
dividerLog('в•җ', 40, { color: '#00cc88' })
```
