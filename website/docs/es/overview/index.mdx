import { Terminal, Paintbrush, ListOrdered, Table, MessageSquare, Activity, Cog, Type, PanelTop, SquareDashed, SplitSquareHorizontal, Bot, TrendingUp, Play, Palette, Package, File, BarChart3, GitCompare, QrCode, Image as ImageIcon } from 'lucide-react'

**DUI** — toolkit de UI para terminal para apps CLI de Node.js. Consciente de ANSI, cero dependencias (solo `string-width`), tree-shakeable.

---

## Inicio Rápido

```bash
pnpm add @bdocs/dui
```

```ts
import { configure, info, success, box, bullet } from '@bdocs/dui'

configure({ prefix: 'mytool' })

info('Iniciando build...')
success('Build completado!')

console.log(box(['Salida lista en dist/'], { title: 'Resultado', style: 'round' }))
console.log(bullet(['Archivo A', 'Archivo B', 'Archivo C']))
```
