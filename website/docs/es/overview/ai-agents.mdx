## Instalación en un solo paso

Ejecuta este comando en la raíz de tu proyecto para instalar la skill de DUI para cualquier agente AI:

```bash
curl -fsSL https://raw.githubusercontent.com/bolt-docs/dui/master/scripts/install-ai-agent.sh | bash
```

El script preguntará qué agente configurar:

| # | Agente | Destino |
|---|--------|---------|
| 1 | **Claude Code** | `CLAUDE.md` (instrucciones del proyecto) |
| 2 | **Claude Skill** | `.claude/skills/dui/SKILL.md` (skill modular) |
| 3 | **Cursor** | `.cursor/rules/dui.mdc` (reglas del sistema) |
| 4 | **OpenCode** | `.opencode/skills/dui/SKILL.md` (skill modular) |
| 5 | **.agents** | `.agents/skills/dui/SKILL.md` & `.agents/skill/dui/SKILL.md` (universal) |
| 6 | **Todos** | Todos los agentes soportados |

### Modo no interactivo

```bash
# Instalar para un agente específico (ej. cursor)
curl -fsSL https://raw.githubusercontent.com/bolt-docs/dui/master/scripts/install-ai-agent.sh | bash -s -- --agent cursor

# Instalar para todos los agentes sin preguntar
curl -fsSL https://raw.githubusercontent.com/bolt-docs/dui/master/scripts/install-ai-agent.sh | bash -s -- --agent all --yes
```

### Desde el repo clonado

```bash
bash scripts/install-ai-agent.sh
```

## Qué incluye la skill

La skill de DUI le da al agente AI conocimiento completo de la librería:

- **API pública completa** — firmas de funciones, tipos e interfaces
- **Ejemplos de uso** — código práctico para cada componente
- **Sistema de temas** — slots de `DuiTheme`, `ColorStyle`, resolución en cascada
- **Mejores prácticas** — comportamiento TTY vs non-TTY, `NO_COLOR`, manejo de errores
- **Configuración** — `configure()`, `getConfig()`, `resetConfig()`

Una vez instalada, tu agente AI sabrá cómo usar `@bdocs/dui` correctamente sin alucinar imports o APIs.

---

## Lo que tu agente AI sabrá hacer

Después de instalar la skill, tu agente AI podrá responder preguntas como:

> "Crea un CLI con un spinner, una barra de progreso y un select usando DUI"

> "Muéstrame cómo personalizar colores con un tema"

> "Construye un asistente de configuración multi-paso con confirmación"

Consulta la [Guía LLM](/docs/es/overview/llm) para plantillas de prompt listas para copiar y pegar.
